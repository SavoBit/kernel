### Buddy Allocator

ref: http://woodz.tistory.com/57

```
struct zone {
	struct free_area free_area[MAX_ORDER/*11*/];
};

struct free_area {
     struct list_head    free_list[MIGRATE_TYPES];
     unsigned long       nr_free;
};
```
